Privacy Policy – Smart-Content-Bot

Last Updated: March 2025

1. Introduction

Smart Content Bot ("the Application") is a private automation tool that interacts with the TikTok API for content posting and trend analysis. This Privacy Policy outlines how this Application handles user data.

2. Data Collection and Usage

This Application does not collect, store, or share any personal user data. All API interactions occur directly between the user’s TikTok account and the official TikTok API.

3. TikTok API Data

The Application uses TikTok's official API to upload videos and analyze content performance. However, it does not store or retain any TikTok user information beyond what is required for API authentication.

4. Security Measures

The Application follows security best practices to ensure API credentials are handled securely. No third-party services or external databases are used to store data.

5. Changes to This Privacy Policy

The developer reserves the right to update this Privacy Policy at any time. Any changes will be reflected on this page.

6. Contact Information

If you have any questions about this Privacy Policy, please contact the developer.
